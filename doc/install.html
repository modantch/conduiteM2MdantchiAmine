Installation

Il faut préalablement installer un ensemble d'outils : un serveur web comme Apache, php et un logiciel de base de données comme mySql . Pour simplifier, nous avons utiliser MAMP sous MAC, WAMP sur windows et XAMPP sur linux. Ensuite nous executons bd_site.sql pour creer la table atelier dans la base bd_site. Ensuite, ouvrez le fichier "Accueil.php", pour débuter sur le site

Pour acceder à la base de données il faut un utilisateur 'root' et le mot passe est 'root' ou le champ est laissé vide '' selon le systeme.
