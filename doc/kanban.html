Rappel des US

  • US1 : En tant que laboratoire je souhaite ajouter un atelier ou lister l'ensemble des ateliers existants
  • US2 : En tant que laboratoire, je souhaite modifier un atelier ou le supprimer

Rappel des tâches

  1. Création de la BD_Site qui contient la table Atelier (id_Atelier, Titre_atelier, theme, type, langue,Niveau, Capacité_accueil, Adresse, durée)
  2. Implementer l’ajout d’un atelier
  3. Ecrire la requete SQL qui permet d’inserer un nouveau atelier dans la table atelier
  4. Créer la page Modifier permettant de modifier les informations d’un atelier, de les enregistrer, de supprimer l’atelier ou bien d’annuler la modification et qui renvoie l’utilisateur sur la page Accueil lorsque l’opération est terminée ou annulée.
  5. Creer u-ne page html/css contenant un formulaire pour la saisi des informations de création d’un atelier. Il faut ajouter à cette vue un boutton « Ajouter atelier »
  6. Implementer un controleur qui recupere les informations saissies sur le formulaire d’ajout et les envoit vers la BD pour exécuter la requete insersion atelier
  7. Tester la fonctionnalité d’ajout : inserer des ateliers Via le formulaire dans la table atelier.
  8. Lister les ateliers
  9. Ecrire la requete SQL qui permet de lister l’ensemble des ateliers de la table Atelier
  10. Creer une page html/css permettant d’afficher la liste des ateliers dans un tableau.
  11. Implementer un controleur qui recupere les données de la table Atelier et les passe à la vue.
  12. Tester la fonctionnalité de : lister des ateliers via la table Atelier.
  13. Modification d’un atelier
  14. Ecrire la requete SQL qui permet de recupere les informations d’un atelier dans la table atelier. Ecrire la requete SQL qui permet de mettre à jour les informations modifiées d’un atelier dans la table atelier.
  15. Creer une page html/css contenant un formulaire qui affiche les informations d ‘un atelier donnant la posibilitè de les modifier.
  16. Implementer un controleur qui recupere les données d’un atelier de la table Atelier et les passe à la vue. Ce meme controleur recupere les informations saisies sur le formulaire apes modification et les envoit vers la BD pour exécuter la requete modifier atelier
  17. Test : - modification d’un atelier et vérifier les modifcations dans la table Atelier.
  18. Suppression d’un atelier
  19. Ecrire la requete SQL qui permet de supprimer un atelier de la table atelier
  20. Ajout d’un boutton de suppression sur la page de modification
  21. Tester la suppression d’un atelier : cliquer sur suprimer et verifier dans la table si l’atelier c’est bien suprimmer.
